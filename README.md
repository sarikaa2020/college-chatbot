College Chatbot:

A college information chatbot built using Python and Flask that answers common student queries using predefined intents.
The chatbot provides accurate responses, includes fallback handling, and can be deployed easily using Docker.

Features:

-> Chat-based interface for college-related queries

-> Intent-based response system using JSON

-> Fallback responses for unknown questions

-> Web interface using HTML templates

-> Docker support for easy deployment

-> Clean and modular project structure

Tech Stack:
| Category    | Technology          |
| ----------- | ------------------- |
| Language    | Python              |
| Framework   | Flask               |
| Chat Logic  | Intent-based (JSON) |
| Frontend    | HTML, CSS           |
| Deployment  | Docker              |
| Environment | Virtualenv          |
